1. Для работы приложения требуется, чтобы через список каталогов, указанный
   в переменной окружения PATH, были доступны следующие программы:
   1.1 java.exe
   1.2 db2.exe
   1.3 db2cmd.exe


2. Перед запуском программы необходимо:
   2.1 Отредактировать файл runme.cmd. В нем нужно задать значения переменных
     - URL - url-базы данных
     - USER - имя пользователя
     - PASSWORD - пароль пользователя
     - DRIVER - имя класса JDBC-драйвера базы данных

     Это необязательно, так как после запуска программа предложит
     отредактировать эти параметры.

   2.2 Отредактировать в файлах viewer_start.sql и viewer_stop.sql строчки,
   относящиеся к соединению и созданию базы данных. В них нужно установить
   необходимые для запуска sql-скриптов имя пользователя, пароль, название
   базы данных. Пользователь может отличаться от того, который указан в файле
   runme.cmd.

3. Программа может работаь с драйверами баз данных:

   - ODBC для IBM Db2          (Класс драйвера sun.odbc.jdbc.JdbcOdbcDriver)
   - ODBC для Microsoft Access (Класс драйвера sun.odbc.jdbc.JdbcOdbcDriver)
   - Postgresql                (Класс драйвера org.postgresql.Driver)
   - IBM DB2 local driver      (Класс драйвера COM.ibm.db2.jdbc.app.DB2Driver)

4. С драйвером COM.ibm.db2.jdbc.app.DB2Driver в программе не работает
   просмотрщик запросов, так как этот драйвер не поддерживает прокрутку
   результатов в произвольном направлении.

5. Переменная окружения CLASSPATH должна содержать список каталогов и архивов
   с необходимыми классами драйверов баз данных.

6. Для запуска программы с тестовым примером надо запустить файл runme.cmd.

7. Для обычного запуска приложения надо запустить команду
   
   java.exe -jar viewer.jar [<url>] [<user>] [<password>] [<driver>]

   или

   java.exe -cp "%CLASSPATH%;viewer.jar" viewer.navigator.Viewer [<url>] [<user>] [<password>] [<driver>]
