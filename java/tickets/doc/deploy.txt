Для размещения приложения на сервере необходимо:

1. Запустить утилиту deploytool.

2. Открыть файл tickets.ear

3. Пометить курсором появившееся приложение.

4. Выбрать пункт меню "Tools/Deploy..." и нажать на кнопку "Finish".

5. Для использования приложения нужно при помощи браузера перейти на адрес
   http://<host>:<port>/Tickets/index.html 
   Рекомендуются браузеры, поддерживающие JavaScript.

6. Для работы в режиме администратора нужно ввести
   имя пользователя "admin" и пароль "admin" (Эти значения можно изменить).

7. Параметры подключения к базе данных находятся в файле tickets.properties.
   В каталоге "$J2EE_HOME/public_html/Tickets/WEB-INF/classes/"
